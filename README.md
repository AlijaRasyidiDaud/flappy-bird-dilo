# Flappy Bird
Created on Unity 2018.1.0f2 <br>
Platform : PC <br>
Genre : Arcade, Hypercasual

## Description
Flappy Bird game with time cycle feature using coroutine.

## Gameplay
Dont crash anything !

## Control
Tap <b>[left click mouse]</b> to flap.
